_Used when no nightly brief is received. The agent works through these tasks
in priority order until morning._

- Check /workspace/tracking/ for any items with status "In Progress" — update if context is available from recent output files

---

### P2 — When Idle

- Review /workspace/context/ files for outdated information and flag for human update
